📖 Sobre
O SK Clock é um relógio educativo e interativo que ensina como falar as horas em inglês usando a hora atual do usuário.
Além de mostrar relógios analógico e digital, a aplicação apresenta a leitura simples, a forma natural em inglês e seu significado em português.
✨ Funcionalidades
Relógios analógico e digital em tempo real
Formas simples e naturais de dizer as horas em inglês
Tradução e explicação em português
Guia de expressões como past, to, quarter e half past
Identificação de AM e PM
Exibição da data atual
Temas claro e escuro com preferência salva no navegador
Interface responsiva para computadores e celulares
🛠️ Tecnologias
HTML5 para a estrutura da aplicação
CSS3 para o design, as animações e a responsividade
JavaScript para o relógio e a geração das expressões
Web Storage API para salvar o tema escolhido
Vercel para publicação do projeto
🚀 Como executar
git clone https://github.com/silaskaleby/sk--clock.git
cd sk--clock
Depois, abra o arquivo index.html no navegador. O projeto não exige instalação de dependências.
